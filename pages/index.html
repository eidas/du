du — Disk Usage Tool

Linux の du コマンドの Windows/Linux 向け CLI ツールです。

ダウンロード

Windows (x86_64) Linux (x86_64)

使い方

du [OPTIONS] [PATH]

Options:
  -d, --depth <N>      表示する階層の深さを制限
  -H, --human-readable  KB/MB/GB 単位で表示
  -s, --summarize       合計サイズのみ表示
  -h, --help            ヘルプを表示

Examples:
  du .                       カレントディレクトリを再帰的に表示
  du -H -d 2 C:\Users        深さ2まで人間が読みやすい形式で表示
  du -s D:\Projects          合計サイズのみ表示
